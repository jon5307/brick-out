# 프로젝트 진행상황

## 1주차 (11/17 ~ 11/23)

* * *

1. 프로젝트 주제 선정
   - 주제 아이디어를 모아서 투표를 통하여 정함. (팀원 전부)
2. 깃허브 저장소 생성
   - 조장(장혁수)가 만들어서 팀원을 초대함

## 2주차 (11/24 ~ 11/30)

* * *

1. 게임 제작
   - 조원(김태훈)이 기본적인 벽돌 깨기 게임을 제작함.
   - 랭킹 시스템을 추가하기위한 클래스 제작(장혁수). 랭킹은 csv파일로 저장하도록 제작함.
     추후 게임 결과를 반영하도록 수정할 예정이다.
1. 의견 제시
   - 조원(홍보석)이 ‘바’ 형식이 아닌 마우스로 진행하는 벽돌 깨기 게임을 만들어 볼 것을 제의함.
